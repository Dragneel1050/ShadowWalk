# ShadowWalk
A python tool for the automation of passive recon


## Useage
Provide API KEYS of the following:

1. Censys.io
2. ZoomEye
3. Shodan
4. SecurityTrails

Once done, use -h to see the available flags.

## Example

python3 shadow.py -u example.com -a
